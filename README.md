# Top Down Survivor

Een kleine 2D top-down Unity game die zichzelf opbouwt zodra je op Play drukt.

## Spelen

- Open deze map als Unity-project.
- Druk op Play.
- Beweeg met `WASD`.
- Mik met de muis.
- Houd de linkermuisknop ingedrukt om te schieten.
- Versla enemies voor XP. Bij elk level kies je 1 van 3 upgrades.

De game spawnt elke 30 seconden een nieuwe wave. Elke wave bevat meer enemies en enemies worden geleidelijk sterker.
