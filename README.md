# Infinite Shooter

Een 2D top-down survivor shooter die zichzelf opbouwt zodra je op Play drukt.

## Spelen

- Open deze map als Unity-project.
- Druk op Play.
- Gebruik het startmenu om een run te beginnen of permanente upgrades te kopen.
- Beweeg met `WASD`.
- Mik met de muis.
- Houd de linkermuisknop ingedrukt om te schieten.
- Dash met `Space`.
- Pauzeer met `Esc`.
- Open permanent upgrades tijdens een run met `U`.
- Versla enemies voor XP. Bij elk level kies je 1 van 3 upgrades.
- Enemies droppen XP pickups en soms health.
- Je verdient upgrade points via waves, bosses en salvage drops.

De game spawnt elke 30 seconden een nieuwe wave. Elke wave bevat meer enemies en enemies worden geleidelijk sterker. Elke 5 waves verschijnt een boss.
